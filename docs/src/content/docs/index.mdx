# ryu_ldn_nx

**ryu_ldn_nx** is a Nintendo Switch sysmodule that enables online local multiplayer (LDN) gaming by connecting to Ryujinx LDN servers. Play "local wireless" games like Mario Kart 8 with friends anywhere in the world!

## Features

- **Easy Installation** - Just extract to your SD card and reboot
- **Transparent Operation** - Games work without modification
- **Tesla Overlay** - Monitor connection status and configure settings
- **Auto Reconnect** - Handles network interruptions gracefully

## Supported Games

Works with games that use Nintendo's LDN service:

- Mario Kart 8 Deluxe
- Super Smash Bros. Ultimate
- Monster Hunter Rise
- Animal Crossing: New Horizons
- And many more...

## Quick Start

```bash
# 1. Download the latest release from GitHub
# 2. Extract ZIP to SD card root
# 3. Reboot into Atmosphere
# 4. Open Tesla overlay (L+DDown+RStick) to verify
```
